# Dippin Dots

* Install homebrew apps
```bash
brew bundle
```

* Make fish default shell
```bash
echo /opt/homebrew/bin/fish | sudo tee -a /etc/shells
chsh -s /opt/homebrew/bin/fish
```

* Fish adding path
```bash
fish_add_path /path/to/add
```

* Ipython settings
```bash
path -> ~/.ipython/profile_default/
```
